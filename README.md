## Проект coffee_report
### Описание
coffee_report - неболшой проект, предназначен для чтения файлов формата csv с данными о подготовке студентов к экзаменам и формирования отчета с выводом в терминал в табличном виде.
На данный момент существует отчет median_coffee, расчитывающий медиану затрат на кофе каждого уникального студента.

> [!IMPORTANT]
> Проверка данных из файлов не проводится. Считается, что файлы всегда валидны. Пример файла ниже
> ![Alt](https://github.com/amiro8256/report_coffee/blob/main/images/example_csv_file.png)
### Как запустить проект
- Клонируйте репозиторий с проектом на свой компьютер. В терминале из рабочей директории выполните команду
```
    git clone ...
```
- Установите и активируйте виртуальное окружение
- Обновите pip
- Установите зависимости
```
    pip install -r requirements.txt
```
### Демонстрация работы
- Создайте тестовые файлы в корне проекта и наполните их валидными данными. Для этого выполните
```
    python manage/create_csv_files.py
```
- Для вывода отчета выполните команду
```
    python report_coffee/main.py -f test_file1.csv test_file2.csv test_file3.csv -r median_coffee
```
Вывод будет такой:
![Alt text](https://github.com/amiro8256/report_coffee/blob/main/images/example_res1.png)
Для обработки собственных файлов укажите полный путь до каждого файла через пробел.

Пример исполнения:
![Alt](https://github.com/amiro8256/report_coffee/blob/main/images/example_res2.png)
### Управление
Управление осуществляется через аргументы командной строки

| Command |  Description |
| --- | --- |
| -f, --files | название файлов или пути до файлов через пробел|
| -r, --report | название отчета |
| -h, --help | посмотреть подсказку |
